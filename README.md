## 📌 Project Overview
This project focuses on using SQL queries to extract 
insights from a real-world e-commerce dataset.

---

## 🎯 Objectives
- Write SELECT queries to retrieve data
- Use WHERE to filter rows
- Use ORDER BY to sort results
- Use GROUP BY with COUNT, SUM, and AVG aggregations
- Use HAVING to filter grouped data
